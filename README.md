# 중간과제 기획서: 인베이더 브레이커 (Invader Breaker)

## 기본 정보
**장르:** 벽돌깨기 (Breakout)  
**기반 템플릿:** 기본 Breakout 코드

## 게임 규칙
- **목표:** 블록의 압박에서 벗어나 최대한 오래 살아남기
- **패배 조건:** 체력(Lives) 전부 소진, 하강하는 블록이 패들과 같은 높이까지 내려옴 (즉시 게임오버)
- **체력 소진 조건:** 공을 놓쳐서 화면 바닥에 떨어뜨릴 경우

## 조작법
- **좌/우 방향키:** 패들(Paddle) 이동
- **위 방향키:** 총 발사
- **스페이스바:** 게임 시작 및 재시작 시 공 투하

## 기반 템플릿과의 차별점
1. **대기 시스템:** 게임 시작 전, 스페이스바를 누를 때까지 공이 화면 상단에서 좌우로 계속 왕복하며 발사 타이밍을 조절함.
2. **공격 시스템:** 패들에서 위로 탄환을 쏠 수 있는 기능 추가. 단, 탄약(Ammo)은 한정되어 있음.(최초 총알 10개 지급)
3. **아이템 시스템:** 블록 파괴 시 일정 확률로 탄약 보충 아이템 드랍.
4. **압박 로직 (인베이더 요소):** 블록들이 고정되어 있지 않고 일정 시간마다 천천히 아래로 전진함. 패들의 위치까지 내려오면 패배함.

## 난이도 설계
**레벨 증가:** 블록을 전부 부술 경우 레벨이 올라가 블록 수가 늘어나고 공 속도와 블록 속도가 빨라짐
